# icon-exporter-For-Blender
A simple extension that allows you to export an image directly to the ".ico" format.  It should work on versions 4.0–4.4; it has not been tested on earlier versions.

Features:

Exports each size as an individual .ico file
Available sizes: 16, 24, 32, 48, 64, 128, 256 px
Full 32-bit RGBA — transparency preserved
Works with Render Result (after F12) — no workarounds needed
Bilinear interpolation when resizing
No external dependencies — pure Python

Where to find it in Blender:

File → Export → ICO (Windows Icon) (.ico)
Image Editor → Image → Export as ICO (.ico)
Image Editor → Image → Save As → Export as ICO
Properties → Output → ICO Export panel
